## 问题/风险

- ZTools 插件中存在直接 Node/Electron API 使用，需要兼容层和权限控制。
- uTools `.asar` 包暂未解包，后续需要 `@electron/asar` 或等价只读解包工具。
- 第三方插件样本只保留在本机，不提交公开仓库。
- 目前只有最小工程壳，还没有托盘、全局快捷键、插件窗口和真实插件运行时。
- 当前插件窗口已注入基础 ZTools/uTools 兼容 API，但还不是完整运行时。
- 兼容桥目前不加载第三方插件自己的 preload，高风险 Node/Electron 能力暂不开放。
- 剪贴板历史、文件能力、截图能力仍是占位。
- PowerShell 执行 npm 要用 `npm.cmd`。
