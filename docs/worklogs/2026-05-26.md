## 交接说明

- release 包仍然被 `.gitignore` 忽略，不提交到 GitHub。
- 如果用户测试 portable 版本失败，先重新运行：

  ```powershell
  npm.cmd run build
  npm.cmd run package:portable
  ```

- 如果 PowerShell 脚本报中文字符串解析错误，保持脚本 ASCII，不要在 `.ps1` 内写中文字符串。
- 截图/钉图目前支持多屏区域选择、综合编辑、标注、复制、保存、钉图、OCR 调用入口、翻译调用入口和操作状态反馈。
- OCR 当前是 PaddleOCR sidecar 适配层；本机已安装到 `%APPDATA%\yang-tools\ocr-runtime`，Python 路径见 `python-path.txt`。
- 翻译当前是统一 Provider 接口；LibreTranslate 默认指向 `http://localhost:5000`，DeepL/OpenAI Compatible 需要在 AI 设置中填写 API Key。
