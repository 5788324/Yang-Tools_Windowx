# 本地插件样本库

这个目录用于保存本机 ZTools/uTools 插件样本，方便后续做兼容分析。

原始插件属于第三方代码，版权和许可证未知，因此本目录内容默认不提交到公开 GitHub 仓库。

当前结构：

```text
local-plugin-library/
  ztools/
  utools/
```

来源：

```text
C:\Users\YANG\AppData\Roaming\ZTools\plugins
C:\Users\YANG\AppData\Roaming\uTools\plugins
```

公开仓库只提交本 README 和分析文档，不提交插件源码或 asar 包。
